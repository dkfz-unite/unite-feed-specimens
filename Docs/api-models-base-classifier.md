# Tumor Classification Data Model
Includes tumor classification data.

_At least one field has to be set_

**`superfamily`** - Specimen tumor superfamily.
- Type: _String_
- Limitations: Maximum length 255
- Example: `"Carcinoma"`

**`family`** - Specimen tumor family.
- Type: _String_
- Limitations: Maximum length 255
- Example: `"Adenocarcinoma"`

**`class`** - Specimen tumor class.
- Type: _String_
- Limitations: Maximum length 255
- Example: `"Colorectal Adenocarcinoma"`

**`subclass`** - Specimen tumor subclass.
- Type: _String_
- Limitations: Maximum length 255
- Example: `"Mucinous Adenocarcinoma"`
